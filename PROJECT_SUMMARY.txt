Change Colors:
 Open: static/css/style.css  
 Find: --accent: #00d4ff;  
 Try: #ff006e (pink), #00ff88 (green), #8b5cf6 (purple)  

Add Projects:
 Open: app.py  
 Find: "projects": [  
 Add new project dictionaries with title, description, tags, github  

Change Animation Speed:
 Open: static/js/three-setup.js  
 Find: .rotation.x += 0.005  
 Adjust number (higher = faster, lower = slower)
